
# **VIDEOGAMES** | Proyecto Individual

Proyecto en el cual se ha utilizado una API para obtener información sobre más de 100 videojuegos. Esta API proporciona datos detallados, como nombres, géneros, plataformas y calificaciones de los videojuegos. El repositorio ha implementado diversas características para mejorar la experiencia de usuario:

Filtrado y Ordenamientos: Se ha implementado la posibilidad de filtrar los videojuegos según diferentes criterios, como género o plataforma. Además, los usuarios pueden ordenar los resultados por nombre, calificación u otros atributos relevantes.

Tarjetas de Detalle: Cada videojuego tiene una tarjeta individual que muestra información clave, como su nombre, género, plataforma y calificación. Al hacer clic en una tarjeta, los usuarios pueden acceder a una vista más detallada con información adicional, como una descripción del juego, año de lanzamiento y desarrollador.

Formulario de Creación de Videojuegos: El repositorio también incluye un formulario que permite a los usuarios agregar nuevos videojuegos a la colección. Los usuarios pueden proporcionar detalles como el nombre, género, plataforma y calificación del juego que desean agregar.
